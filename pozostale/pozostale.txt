data-atrybuty  <- bardzo potezne narzedzie 

pozwala nam pozbyc sie ID

<img src="" alt="">    <-zwykly atrybut
<div data-number="'23" data-more-info="to jest tekst"></div>  <- data atrybuty


const div = document.querySelector('div')
console.log(div)

const div1 = document.querySelector('[data-number="23"]')   
console.log(div1)

console.log(div1.dataset)  <- zwrocil nam obiekt!!!
console.log(div1.dataset.number)   <- zwroci nam 23
console.log(div1.dataset.moreInfo) <- zwroci nam wartosc atrybutu more info

div1.dataset.color = 'blue'  <- stworzylismy data-atrybut o nazwie color z wartoscia blue


mozna pracowac ze wczesniejszymi metodami:

div1.setAttribute('data-test', '123')
console.log(div1.dataset)
DOMStringMap(3) { number → "23", moreInfo → "to jest tekst", test → "123" }
